# InAmigos Foundation — Awareness Webpage

This repository contains a single-file HTML/CSS awareness webpage for InAmigos Foundation built as a professional, responsive NGO landing page.

## About
- Project: InAmigos Foundation awareness site
- Tagline: "Uniting Minds For Change"
- Founder: Govind Shukla (founded 23 September 2020)
- Registrations & Certifications: Section 8 NGO, CSR-1, NITI Aayog, ISO 9001:2015

The content and structure were created after reviewing the official website and public social media presence (Instagram: @inamigos, LinkedIn: InAmigos Foundation, Facebook posts using #InAmigos) and written to summarize the organisation's programs and impact without copying site text verbatim.

## Features
- Modern responsive layout using CSS Grid & Flexbox
- Green + blue NGO color palette
- Hero with CTAs (Join, Volunteer, Donate)
- About, Stats, Projects, Impact, Events, Gallery, Testimonials, Volunteer/Internship, Contact sections
- Smooth micro-interactions and hover effects
- Font Awesome icons (via CDN)

## Files
- `inamigos.html` — single-file HTML with embedded CSS (open directly in a browser)
- `README.md` — this file

## How to view
1. Open `inamigos.html` in any modern browser (Chrome, Edge, Firefox).
2. No build step required; the file is fully static.

## Deploy (suggested)
- Use GitHub Pages: in the repository settings enable Pages from `main` branch (root). The site will be available at `https://<username>.github.io/<repo>`.
